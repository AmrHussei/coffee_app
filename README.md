# coffee_app

 Flutter e-commerce application about coffee ordering. Main goal of which is to gain more expertise in mobile cross-platform app development.
 
 # technologies
 - Flutter 
 - Dart
 - firebase auth
 - Rest API
 - GetX
 - http 
 - geocoding 
 - geolocator
 
 # Architecture
 
   MVC Design Pattern with GetX
